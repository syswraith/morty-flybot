FLY

What is FLY bot?

FLY is a NodeJS-based, all-rounder moderation bot with fun commands built in it. It is a great entertainment for your server.

How to use it?

Using FLY is very easy. All you have to do is invite the bot to your server by clicking here.

Is it open-source?

Yes! I built it on Replit from scratch, referring to various tutorials and code snippets. You can find the source-code here. I will shortly be posting it on GitHub so it's easier for everyone to suggest and make edits to the code.

WARNING!

Because FLY is still in development it may stop working. If this happens, we will immidiately respond. If it crashes and does not come online after 5 mins, please consider raising a complain here. We will try to sort it out as soon as possible.
